numbers = [1, 8, 7, 6, 3];
console.log(numbers.reduce((p,n) => p + n ,0));

파라미터(p,n) => p + n = 콜백
0 = 초기값 지정

1. 초기값 0이 p에 입력
2. numbers array의 첫번째 값인 1인 n에 입력
3. p + n = 0 + 1의 결과값 1 반환
4. 3에서 반환한 값(1)이 p에 입력
5. array의 두번째 값인 8이 n에 입력
6. p + n즉 1+ 8의 결과값인 9가 반환
7. 6에서 반환한 값(9)가 p에 입력된다
8. numbers 리스트의 모든 값들을 다 순회할때까지 반복 -> 모든 값을 다 더한 25반환
